# ComfyUI IG4 Solo

A custom guider node for ComfyUI, providing optimized conditioning modes to give you finer control over your sampling process.

## Features
* **Optimized Modes:** Toggle between `cond_only`, `minimal_cfg`, and standard `cfg`.
* **Staged Guidance:** Perfect for use with `CFGOverride` nodes to create natural-to-sharp sampling transitions.
* **Lightweight:** Designed to be efficient and clean.

## Installation
1. Open **ComfyUI Manager**.
2. Click **Install via Git URL**.
3. Paste: `https://github.com/PartisanoHub/ComfyUI-IG4-Solo`
4. Restart ComfyUI.

## Usage
Connect the `GUIDER` output to your Sampler node. Pair this with `CFGOverride` for advanced staged guidance scheduling.
